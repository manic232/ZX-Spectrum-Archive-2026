Manic Mixup

 

Created By: Andy Ford, Ian Rushforth
Release Year: 2018

Manic Mixup is a 2018 remix of the classic ZX Spectrum game Manic Miner, developed by Andy Ford and Ian Rushforth. It serves as a sequel to the authors' previous game, Jet Set Mini, and blends two earlier projects by Dr. Andrew Broad: the laterally-inverted reniM cinaM and his 2008 remix of Manic Miner. 

The game features an updated storyline where Miner Willy returns to the subterranean realm to address financial difficulties caused by decades of partying. Key updates include overhauled caverns, enhanced in-game music, and rewritten game engine routines that introduce new sound and visual effects, additional gameplay elements like a hard hat mechanic, and bug fixes while maintaining the original 8-bit aesthetic.
