Horace to the Rescue

 

Created By:  Steve A.J. Broad
Release Year: 2013

My Rendition of Horace to the Rescue is a 2013 homebrew arcade game for the ZX Spectrum 48K, created by Steve A.J. Broad (UK). Inspired by the classic game Hungry Horace, it is a fan-made "rendition" designed to look and feel like an authentic 1980s Melbourne House release. 

The game was developed using the HiSoft COLT Compiler. It features action-platforming gameplay where the player controls Horace to collect coins and save a bell, mimicking the style of the original Horace series. While the original Horace to the Rescue was a planned but unreleased game by Melbourne House, Steve Broad's version serves as a tribute to what the sequel might have been.
