Help Using this Site

I originally designed this site to be viewed in Internet Explorer on a Windows 98 or Windows XP PC. The layout looks best at 1024×768 or higher, and while the site will display perfectly well in modern browsers, a few formatting quirks do appear here and there.

For the most authentic and consistent experience, I recommend viewing the site with RetroZilla. It runs flawlessly on modern versions of Windows as well as Windows 98, and the site looks exactly as intended when using it. On Windows 98 you can still use IE5 (IE6 works too, though with a few minor issues), but RetroZilla remains the best choice for consistency across operating systems.

The area at the top of the page is the main navigation menu. It lets you jump to the primary sections of the site. (See About for more details.)

The left pane changes depending on where you are. In most sections it contains links to the relevant game reviews or instruction pages. In the Archive Home area, it shows various Spectrum-related web links — including several new ones added in 2026.

I’ve tried to keep things consistent throughout the site. Regular internal links are white and underlined, and those pages include a Back button at the bottom to return to the previous level. External links are Cyan and open in a new window.

At the bottom of each main section you’ll find links to additional material such as interviews, history, and more.

All game review and instruction pages now include box art, which you can click to enlarge and click again to return. Each game page also has a “Play Game” option — a small tape icon at the top right — which opens the game’s ROM in your default Spectrum emulator. I use Spectaculator, but any emulator should work as long as you’ve associated TZX, TAP, DSK, and Z80 files with it. Most ROMs are in TZX format, though a few use TAP, DSK, or Z80.

 

Notes on Playing the Games

Spectrum games — especially those released in the later years — often came on two or more cassette tapes. Many titles required you to insert a different tape, or flip the tape over, to load another section of the game. Because of this, Spectrum ROM sets frequently include multiple files: Side A, Side B, Tape 1, Tape 2, and so on.

Each game page on this site includes only one ‘Play Game’ link. I didn’t want to clutter the page with multiple links for every tape required to complete a multi‑load title, so in most cases I’ve included just the first side or tape. If you want to play multi-tape games properly, it’s best to source the full ROM set manually. Think of the ROM links here as a Brucie bonus!

Many of the games listed — particularly in the CRASH Smash section — are 128K titles. Spectaculator remembers the last model you selected, so if a game refuses to load, try switching between 48K and 128K modes and reload the game.

If you’re viewing this site on a Windows 98 PC or 86Box, Spectaculator can’t be used for obvious reasons. You can still use ZX32, but compatibility isn’t perfect: some games will work, some won’t, and DSK images won’t load at all.

I’ve tested all 356 ROMs to make sure they are the correct games and that they load properly. Of course, I haven’t played them to completion — can you imagine how long that would take — but I’ve tried to ensure that everything loads without drama. A handful of titles still have copy protection, and I wasn’t able to find alternative ROMs without it. Thankfully, it’s only one or two.
