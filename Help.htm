Help Using this Site

I originally designed this site to be viewed with Internet Explorer on a Windows 98 or XP PC.  The site looks best at 1024x768 or higher, and whilst it will display on modern browsers, there are formatting glitches in some areas.  I would recommend that you view this web site with RetroZilla, this browser runs perfectly on modern Windows as well as Windows 98, and the site looks great with it. On Windows 98 you can use IE5 (IE6 has some minor issues), and while it looks fine, I would still recommend RetroZilla for consistency across operating systems.

The area at the top of the page, is the main navigation menu.  It is used to jump to the seven main sections of the site, See About for more.

The left pane changes depending on where you are.  It contains links to game reviews or instructions that the section relates to, in the Archive Home area it has various Speccy related web links, including some new ones I added in 2026.

I have tried to be consistent throughout the site, if you click on a regular link, underlined in white there will be a Back button at the bottom of the page to go back a level. External links, are coloured Cyan, and open in a new window.

At the bottom of each main section, you will find links to further material, such as interviews, history and more.

Game review and instruction pages all now contain box art that can be clicked on to expand the image, you can click again to return to the previous page. There is also now a 'Play Game' option on every game page (A tape at the top right) that open's the game's ROM in your default Spectrum Emulator, I use Spectaculator, but other emulators should work, as long as you set tzx, tap, dsk and z80 files to open with them. Most of the ROMs are in TZX format, but one or two are tap, dsk or z80 files.

Some notes on playing the games:

Spectrum games (particularly in the latter years) often came on two or more cassette tapes, often games would require you to insert a different tape or turn the tape over to load in a different section to the game. Because of this, Spectrum ROM images often have more than one file, Side A, B or Tape 1, 2 etc. There is only one link per page for each game here, I do not want to have multiple 'Play Game' links to open the different tapes required to finish the game, so if a game is a multi-tape title, I have generally just included the first side or tape. If you really want to play these game properly then best to source the ROM files manually, think of the ROM links as a Brucie bonus feature!

Many games listed (mainly in the CRASH Smash section) are 128k, the Spectaculator emulator remembers the lest model you selected, so if you find that a game won't load, try switching models (48k or 128k) and reloading the game.

If viewing this site on a Windows 98 PC or 86Box, the Spectaculator emulator cannot be used for obvious reasons. You can still use ZX32, but compatibility will not be as good, some game will work, some won't and dsk images will not work at all.

I have tested all 356 ROMs and made sure that they are the correct game and load properly, obviously I have not played them to completion (can you imagine how long that would take?). I have tried to make sure that all games load without any drama, however I did notice that some have copy protection, and I was not able to find an alternative ROM without it. There are not many, just one or two.
