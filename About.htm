About This Site

In January 2003 I stopped working on this site. I got bored with the tedious process of completing the CRASH Smash section. I was sourcing instructions for each game, and then creating a page for it. I stopped at Popeye and for the next 23 years, left it at that. Fast forward to April 2026 and I made the decision to finish off what I started all those year ago. Back then I would have been using Windows XP and I created these pages using FrontPage 2000. Amazingly it still runs on Windows 11, so I was able to complete the site using the same software.

That is not all I have done though, I have made numerous updates. In 2001 the main incentive for creating this site was to accompany a folder I created full of Speccy games and box art, all sorted nicely into folders that matched the games listed here, however that folder has long since been deleted, so I thought that the best approach for this update would be to have the box art and a playable game link integrated into the game info pages, rather than having to open an external folder. I decided to have the game lists linking to reviews from CRASH online rather than instructions (mainly because it was easier than processing the instructions if I am honest).  I wanted this functionality in 2001 when I started the web page, but I did not have the patience to add box art and game links to 356 pages!, the only reason I have done it now is largely thanks to AI. Copilot has been an essential tool for me in working on this reboot, I used it to create a number of PowerShell scripts to automate much of the work, and I could not have achieved any of the laborious repetitive work without it.

I have loved the ZX Spectrum ever since owning my first 48k Spectrum back in the early eighties.  It was brought for me by my parents one Christmas, I can remember the first game I played on it on that Christmas night, I was banished to the dining room and loaded up Horace Goes Skiing, my Speccy came with the game bundled in.

The ZX Spectrum holds special nostalgic place in my heart, many of the games are still fun to play today, albeit with cheat pokes to make them easier! Alien 8 was a favourite of mine, my school friend (Michael Brown I think was his name) copied it to a tape for me!.  Speccy games were intentionally hard, developers only had a limited amount of memory to play with, so the developers compensated by making them hard in the hope they would take longer for the player to complete.  Back then game play was the most important consideration, graphics were not the main focus like they are today.  That is the reason the Spectrum has the following it has, the games are still very playable, and was the main inspiration for me creating this site and collecting games.

After discovering eBay back in the early 2000s, and seeing that I could get hold of a second hand Spectrum for about £20-£25, I wasted no time in bidding for one.  As well as buying a Spectrum, I also collected games and own a number of Ultimate titles.  Seeing those loading screens after all those years brought fond memories.

These pages were started by me in December 2001, abandoned in January 2003 and finally completed in April 2026. I used to use the ZX32 Spectrum emulator to play games, but these days I use Spectaculator or Retro Virtual Machine. ZX32 can still be used when viewing the site on a Windows 98 PC or 86Box though, but compatibility is not as good.

Reviews, Instructions and a ton of other information can be found within these pages, as well as history, interviews, web links and loads more.  There are several sections to the site.  They can be accessed via the main menu at the top of the page.

  • ZX Spectrum Archive Home: This is the home page, it has information on the ZX Spectrum, as well as history, facts, figures and web links.
  • CRASH Smashes: Contains a complete list and reviews for the games that were awarded a 'CRASH Smash' by everyone's favourite Speccy mag, CRASH.
  • Ultimate Play The Game: This is an area concentrating solely on Spectrum games written by Ultimate Play The Game.
  • The Toy Box: Here I have attempted to bring together some of the games I remember playing as a child.
  • Pokes One/Two: Thousands of Pokes, Infinite lives, maximum energy etc.
  • About: The page you are viewing now.
  • Help: General help area for using this site.
