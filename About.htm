About This Site

In January 2003 I stopped working on this site. I’d grown bored of the tedious process of completing the CRASH Smash section — sourcing instructions for each game, creating a page for it, and repeating that cycle endlessly. I stopped at Popeye, and for the next 23 years… that’s where it stayed.

Fast-forward to April 2026, and I finally decided to finish what I’d started all those years ago. Back then I was using Windows XP and building these pages in FrontPage 2000. Amazingly, it still runs on Windows 11, which meant I could complete the site using the exact same software I began with.

But I didn’t just finish it — I updated it.

In 2001, the main reason for creating this site was to accompany a folder I’d built full of Spectrum games and box art, all neatly sorted into directories that matched the lists here. That folder is long gone now, so for this reboot I decided the best approach was to integrate the box art and a playable game link directly into each game’s info page, rather than relying on external folders.

I also chose to link the game lists to CRASH Online reviews rather than instructions — partly because it’s cleaner, and partly because, if I’m honest, it was far easier than processing hundreds of instruction text files.

I wanted this functionality back in 2001, but I simply didn’t have the patience to add box art and game links to 356 pages. The only reason I’ve managed it now is largely thanks to AI. Copilot has been an essential tool throughout this reboot — helping me generate PowerShell scripts, automate huge chunks of repetitive work, and tackle tasks that would have been impossible (or at least soul-destroying) to do manually.

My Spectrum Story

I’ve loved the ZX Spectrum ever since owning my first 48k Spectrum back in the early eighties. My parents bought it for me one Christmas, and I still remember the very first game I played that night. I was banished to the dining room, loaded up Horace Goes Skiing that was bundled with the machine — and that was it. I was hooked.

The Spectrum holds a special nostalgic place in my heart. Many of the games are still fun to play today, even if I now rely on cheat pokes to make them a bit kinder. Alien 8 was a favourite of mine; my school friend — Michael Plumstead, copied it onto a tape for me. Speccy games were intentionally hard. With so little memory to work with, developers compensated by making them challenging so they’d last longer. Gameplay was everything back then; graphics were secondary. That’s a big part of why the Spectrum still has such a loyal following — the games remain genuinely playable — and it’s what inspired me to create this site and collect the games in the first place.

When I discovered eBay in the early 2000s and realised I could buy a second-hand Spectrum for about £20–£25, I didn’t hesitate. Along with the machine, I started collecting games again and even picked up several Ultimate titles. Seeing those loading screens after all those years brought back a flood of memories.

I began building these pages in December 2001, abandoned them in January 2003, and finally completed them in May 2026. Back then I used the ZX32 emulator to play games; these days I use Spectaculator or Retro Virtual Machine. ZX32 still works if you’re viewing the site on a Windows 98 PC or 86Box, but compatibility isn’t as strong.

Within these pages you’ll find reviews, instructions, history, interviews, web links, and plenty more. The site is divided into several sections, all accessible from the main menu at the top of the page.

  • ZX Spectrum Archive Home: This is the home page, it has information on the various ZX Spectrum models, history, facts, figures and more.
  • CRASH Smashes: Contains a complete list and reviews for the games that were awarded a 'CRASH Smash' by everyone's favourite Speccy mag, CRASH.
  • Ultimate Play The Game: This is an area concentrating solely on Spectrum games written by Ultimate Play The Game.
  • The Toy Box: Here I have attempted to bring together some of the games I remember playing as a child.
  • Pokes One/Two: Thousands of Pokes, Infinite lives, maximum energy etc.
  • About: The page you are viewing now.
  • Help: General help area for using this site.
