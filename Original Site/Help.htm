Using This Site

This site makes extensive use of frames.  The frame above is the main menu frame and has links to the different sections that are discussed thought this help area.  I have designed these pages to  be used with Internet Explorer 4 and above and have not tested them with Navigator.  They will look best with a screen resolution of 1024x768.

There is a folder icon at the top of each section that opens the folder that the section relates to.

The left had frame changes depending on which section you are looking at.  It will have links to the various game instructions that the section relates to, so for example on the CRASH Smash section the left hand frame will have links to all the CRASH Smash game instructions.  The exception is the Archive Home area, this section's left hand frame has various Spectrum web links.

The frame that always sits at the top of the site, is the main menu frame.  It is used to navigate to all the different sections of the site, please see About for help on what these sections contain.

I have tried to be consistent throughout the site, so for example if you click on a local link (a link that takes you to a page within the archive, rather than another site) there is always a Back button at the bottom of the page, that will take you to the page you were last looking at.

At the bottom of each main section, you will find links to further material, such as interviews, history and even a few relevant game reviews from the pages of CRASH.

Links to other pages contained within the confines of these pages are underlined in White Like This, these links can be seen without being connected to the Internet, because they are a local link (the pages are stored on the disc or CD Rom).  They will usually open in the main frame you are looking in now.  If a link is going to take you to another web site, outside of my archive pages, then it is in Italic underlined Cyan Like This.  I have made these links open in a new window (with the exception of the web links in the Archive Home frame) for ease of navigation.  Obviously an open Internet connection must be active.  Rare email links might pop-up, they are italic underlined Yellow Like This.

White underlined italic links Like This are direct links to the folders in the Archive, so if for example you click on Games - Archive, a new page will open with the contents of that folder.

Links to game instructions in the left hand frame are not underlined, for purely aesthetic reasons.

I have included a few hidden links in the site (try clicking on a few pictures and see what happens ;o)

You will have noticed that the Ultimate & My Games Instructions have a nice cover image on each page, but the rest don't, the reason is mainly down to time constraints.  I made the effort for Ultimate & My Games because they are special, but I figured that I would leave the rest as plain text.

 

Archive File Folders

Each folder in the archive has in it the games detailed in the corresponding link in the main menu frame above, with the exception of the Games - Archive folder (see below for explanation).

Archive Home, Pokes One & Two, About and this help section do not have corresponding folders, as there are no files or games for these areas.

Below is a description of each folder's contents...

  • Games - Archive: Contained in this folder are all the games that are not part of the main focus of this archive (so, pretty much every game that was not a CRASH Smash, one of the games I owned as a child, an Ultimate title or Spectrum Utility).  There is no real mention or indeed help of any description for the games in this folder in these pages, due to the fact that there are so many games in it, that creating nice HTML pages to support them would be too laborious a job to take on.  Help files for most of the games in Games - Archive can be found in Games - Archive\Extras\Instructions.
  • Games - Crash Smashes: Contains games that received a 'CRASH Smash' review award by that much loved Spectrum mag of the 80's & early 90's, CRASH.  There are missing titles from the Archive, click here for more.
  • Games - My Games: Contains all the games I remember owning as a child.
  • Games - Ultimate Play The Game: Contains all the Ultimate Play The Game titles.

In each of the above folders there is a sub folder called Extras.  Here you will find such wonders as Cassette Inlays, Adverts, Maps, Photos and just about anything else that is related to the games contained therein.  Most of the material here was sourced from the WOS FTP Area or WOS CD Rom.

There are a few duplicate files, For example there are a few Ultimate titles in the Games - CRASH Smashes folder as well as the Games - Ultimate Play The Game folder, this is because they were a CRASH Smash game, as well as being an Ultimate title.  Also pretty much every file in Games - My Games is duplicated from another folder, again because most of the games I owned were either a CRASH Smash, Ultimate title or just one of the ones in the Games - Archive folder.  I won't bother to detail the rest, just bare in mind that they are there.

 

The ZX32 Emulator

I use the splendid ZX32 emulator to play the games in my collection.  It is by far the best of It's type I have come across, mimicking the various incarnations of the Spectrum (16K, 48K, 128K, +2 & +3) perfectly.  I have tried a lot of other Spectrum emulators (there are around 27 to the power of 16 Spectrum emulators to be found on the Internet!!!), but none work as well or as easily as ZX32.  It will even crash certain games that a real Spectrum would!  Games can generally be played simply by double clicking on the game file you want to play, ZX32 will then load and start the game.

The vast majority of games are in the .tap format, because .tap images are my personal favourite file format.  .tap files can be loaded in real time by choosing the appropriate option in ZX32 (hit F9, choose Hardware and move the tape slider right down to Emulate the loading process accurately).  Doing this takes longer to load the game, but for pure retro enjoyment it can't be beaten.  A few files are in .sna format (memory images), because some games do not work as .tap files.

Pokes can be used by first loading the game you want to play, going to the ZX menu and choosing Poke Memory...  Find the poke you want using Pokes One or Two, then simply enter the correct value in the boxes provided.  Please bare in mind though that there are probably a lot of pokes that do not work.  I have not tested them (and have no intention of doing so).  If you try a poke in Pokes One and it does not work, try Pokes Two, if it still does not work, well... tough!!

Hitting F4 while playing a game will put the emulator into a nice full screen mode that looks exactly like you are playing on a TV screen, you even get the borders.  Hitting Alt+F1 will bring up an image of the 48K Spectrum Keyboard.

Please read the ZX32 help file for further assistance, and for goodness sake get playing... NOW!!
