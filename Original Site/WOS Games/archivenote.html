ACCEPTABLE USE POLICY STATEMENT

Due to ongoing abuse of this server, very strict access rules apply, all of which are automatically enforced:
  • The maximum allowed number of simultaneous sessions is 3.
    If you use a download manager, configure it properly first, as most of them have a default number of simultaneous sessions (or 'connections') of at least 4.

  • The maximum session time is 2 hours.
    You can start new sessions after that time, so long as the daily maximum number of downloads has not been reached yet.

  • The maximum number of downloads per session is 250.
    The maximum number of downloads per 24hr is 1000.
Exceeding sessions are automatically blocked and abusers are banned for a day. Persistent abusers are banned forever and reported to their ISP.
  • You are explicitely discouraged to draw full copies of the entire archive. If you want everything, you can buy the CD-ROM of the archive, courtesy of Craig Rothwell.
    More info on this very cheap CD-ROM can be found at
        http://www.worldofspectrum.org/cdrom.html
    Special precautions are taken to stall leechers. Persistant leechers are banned from the archive.

  • All (trans)actions are logged. The logs are kept for a week. The information is kept confidential (except for abuse; see below!).
Note that all abuse is publicly listed realtime, along with your IP address, and banned IP addresses may be exchanged between servers on the Internet for extended blocks, so please behave. We really have a zero-tolerance policy!

If you don't know how to handle each file format, please see the File Formats page.
