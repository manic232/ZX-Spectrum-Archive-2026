ARCHIVE - LEGEND
Lines in bold point to ``perfect originals'' (cannot be improved)

The game name endings have the following meaning:

(demo) : It's the demo of the game
(error) : The snapshot has errors in it (probably crashes)
The column 'S' means: the game contains the original loading screen
The column 'O' means: the zipfile contains the original loading screen as .scr file
The column 'F' means: the zipfile contains an info file
The column 'P' means: the zipfile contains a pokes file
The column 'I' means: the snapshot could be improved
The column 'GameSz' gives the length of the gamefile
The column 'ZIPSz' gives the length of the zipfile
