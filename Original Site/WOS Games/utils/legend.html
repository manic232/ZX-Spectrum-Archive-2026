TZX files also report the encoding type if special:

(AL) = Alkatraz
(BL) = Firebird's BleepLoad
(DI) = Digital Integration scheme
(DL) = Dinamic Loader
(PO) = Paul Owens Protection System
(PW) = Power-Load
(S1) = SpeedLock 1
(S2) = SpeedLock 2
(S25) = SpeedLock 2.5
(S3) = SpeedLock 3
(S35) = SpeedLock 3.5
(S4) = SpeedLock 4
(S5) = SpeedLock 5
(S6) = SpeedLock 6
(S7) = SpeedLock 7
(SE) = Search Loader
(SL) = SoftLock
(UL) = Elite Uni-Loader

The game name endings have the following meaning:

(demo) : It's the demo of the game
(speech) : It has support for the Currah uspeech interface
(error) : The snapshot has errors in it (probably crashes)
(+TAP) : The following levels are in a TAP file
(+VOC) : The following levels are in a VOC file
(-128) : Originally a 128K game, but snapped at 48K
(+128) : Originally a 128K game, snapped at 48K, but convertable

The column 'GameSz' gives the length of the gamefile
The column 'ZIPSz' gives the length of the zipfile
