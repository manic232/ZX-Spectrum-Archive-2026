PS-ZX (Majik)

  1. Emulates the 48K machine (with Hungry Horace pre-loaded).
  2. No load capability.
  3. No save capability.
  4. No real tape support.
  5. Requires a Yaroze (programmable Playstation).
  6. Speed?
  7. Freeware?
  8. Good points?
  9. Plays only Hungry Horace. No sound. Available only to Net Yaroze members.
  10. I don't have a Playstation, so I haven't actually tried this. Comments from anyone who has would be welcomed.
  11. Available from the author's Net Yaroze home page (You need to be a Net Yaroze member to access this). Information available from the author's PS-ZX Page.

Speculator v0.9b (Gabriele Roncolato)

  1. Emulates the 48K machine.
  2. Loads from a set of .SNA files in the executable (it is possible to change these by patching the executable). The next version should be able to load games from CD.
  3. No save capability.
  4. No real tape support.
  5. Requires a Playstation, a Datel Pro Action Replay with Comm Link interface and cable, and the EZ-O-RAY action replay rom replacement (available here). The next version should work on a standard Playstation when burned to a CD.
  6. Runs at true Spectrum speed.
  7. May be freely distributed for no commercial purpose in its original archive.
  8. Good points?
  9. Undocumented flags (used by Sabre Wulf and others) and R register not emulated.
  10. Again, I don't have a Playstation, so I haven't actually tried this. Comments from anyone who has would be welcomed.
  11. Available from the author's home page or World of Spectrum.
